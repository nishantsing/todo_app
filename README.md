# Todo & Notes

### Steps to Create Todo
	* Login
	* Click on bottom right icon
	* Add TodoName and the Todos
	* Click Submit

### Steps to Create Notes
	* Login
	* Click on bottom left icon
	* Add NoteName and the Notes
	* Click Submit
